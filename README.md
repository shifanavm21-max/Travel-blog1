# 🌍 WanderWorld - Travel Blog Website

Welcome to **WanderWorld**, a creative travel blog website designed to showcase the beauty of Kerala’s top tourist destinations.

## ✨ About the Project
This project is a simple and responsive travel blog webpage developed using HTML and CSS. It highlights popular places like Munnar, Athirappilly, and Alappuzha with an attractive layout.

## 🏞️ Featured Destinations
- 🌄 Munnar – A beautiful hill station with tea plantations
- 🌊 Athirappilly – Famous for its stunning waterfalls
- 🚤 Alappuzha – Known for backwaters and houseboats

## 💻 Technologies Used
- HTML5  
- CSS3  

## 📱 Features
- Responsive design (mobile-friendly)
- Clean and modern UI
- Destination cards with images
- Simple navigation bar
- Contact form section

## 🚀 How to Run the Project
1. Download or clone the repository  
2. Open the project folder  
3. Run `index.html` in any web browser
